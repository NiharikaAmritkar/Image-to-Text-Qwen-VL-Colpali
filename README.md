### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
3. Run the Jupyter notebook if the app does not process (run on kaggle GPU P100)
